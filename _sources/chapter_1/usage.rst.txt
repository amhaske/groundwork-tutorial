.. _usage:

Usage
=====

1. call ``groundwork``
2. Explain help message
3. Call ``groundwork docs``
4. Explain what doc is
5. Show plugin_class and explain. Show plugins and explain.
6. Explain, why dynamic documents are awesome
7. Explain, documents exist in memory and it can have different viewer (cli, writer, html, ...)
8. Call ``groundwork doc_write``
9. Call and explain ``groundwork doc_write --help``
10. Call ``mkdir temp`` and ``groundwork doc_write -h temp``
11. Show temp folder content and open one html file in browser
